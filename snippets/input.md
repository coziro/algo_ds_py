# Input

## One Variable
string
```python
S = input()
```
int
```python
N = int(input())
```

## Two Variables
int
```python
N, M = map(int, input().split())
```

## Any Number of Variables
int
```python
A = list(map(int, input().split()))
```

## 2-D List
string
```python
A = [input() for _ in range(H)]
```
